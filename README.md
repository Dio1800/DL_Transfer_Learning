# DL_Transfer_Learning
Using Python, perform transfer learning to train VGG16 model from keras to learn an accurate image classifier from a new small dataset

## Challenge
The challenge consists of applying the Transfer Learning method in a Deep Learning network in the Python language. 

1 - Download cats_vs_dogs MNIST dataset from TensorFlow https://www.tensorflow.org/datasets/catalog/cats_vs_dogs?hl=pt. 

2 - The page will have a link to image files https://www.microsoft.com/en-us/download/details.aspx?id=54765. The Dataset encompasses two classes: cats and dogs. 

## Output
The output is a github repository with at least 3 files:
1 - README - this file with a small explanation.
2 - transfer_learning.ipynb - The jupyter file with sections for python code and explanations markdowns
3 - .gitignore - A configuration file to prevent uploading images and useless files to repository.
